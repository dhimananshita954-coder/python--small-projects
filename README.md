# python--small-projects
This repository containssmall  python gui projects like calculator , student management system and marks prediction that created for learning and practice. these projects demonstrate the use of python, 
tkinter and basic machine learning .
##PROJECTS INCLUDED
### 1. Calculator GUI
A simple calculator application built using python tkinter which includes different mathematical operations on numbers such as addition, subtraction, division and multiplication which are basic operations.

### 2. Student Management System
A GUI based application to manage student details and records, where it includes adding , viewing and deleting students and their details from the recod or list.

### 3. Marks Prediction using Machine learning
A python Gui project that predict student marks basedd on study hours-
.uses linear regression from scikit learn
. built with tkinter for the interface

##Technologies used
